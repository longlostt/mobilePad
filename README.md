# Get started

## Prerequisites (to be installed on your PC separately, not with npm)
MongoDB  
Node.JS

## Installation
```
npm install
npm i -g nodemon (for real-time updates)
npm run dev
```
### Also,
To run mongoDB:
```
mongosh
show dbs
```
### View at http://localhost:3000/

## Stack
### Node.js, MongoDB, Express.js
